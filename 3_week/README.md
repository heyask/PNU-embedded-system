## 1.2 세부실험내용

USER S1: PD2, PD3 LED 번갈아가며 점멸(PD4, PD7 LED 꺼짐)
USER S2: PD4, PD7 LED 번갈아가며 점멸(PD2, PD3 LED 꺼짐)
Select: 모든 LED off

4. RAM에 올려 정상적인 동작 유무 확인
